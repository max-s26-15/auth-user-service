# Start prod

#### Build containers
```
yarn services:build
```

#### Start containers
```
yarn services
```

#### Remove containers
```
yarn services:clean
```

# Start dev
#### Build containers
```
yarn dev:services:build
```
#### Start containers
```
yarn dev:services
```

#### Start auth service
```
yarn start:debug
```

#### Remove containers
```
yarn dev:services:clean
```
